# ☁️ Prediksi Cuaca Indonesia – Machine Learning + OpenWeather API

Aplikasi Streamlit interaktif yang menampilkan data cuaca real-time dari OpenWeather
dengan tampilan modern dan prediksi suhu menggunakan Linear Regression.

## Fitur
- Prediksi cuaca otomatis berdasarkan 41 kota besar di Indonesia
- Tema modern dan minimalis
- API otomatis dari OpenWeather
- Ekspor hasil prediksi ke CSV

## Jalankan Lokal
```bash
pip install -r requirements.txt
streamlit run prediksi_cuaca_app.py
```
